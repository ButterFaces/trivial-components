Configuration Options

        

            

                
                

                    Option
                    Description
                    Default
                

                
                
                

                    valueProperty
                    Name of the property which should be submitted on form submit. If
                        null, then the content of the text input will serve als value.
                    
                    'displayValue'
                

                

                    allowFreeText
                    Allow free text entry. If
                        true, the user is not forced to select one of the suggested entries but can type whatever he wants into the field. If the user leaves the field without selecting any of the suggested values, then his input remains visible. If
                        false, the user input will be deleted if he does not select a suggested entry.
                        However, to acutally use the free text input string, you need to set valueProperty to null, so the free text gets written into the original input. (We will probably change this and make getSelectedEntry() return a fake entry with the inputTextProperty set.)
                    
                    false
                

                

                    inputTextProperty
                    Name of the property which will be filled into the input field when the user selects an entry. This property is also used for auto-completion.
                    'displayValue'
                

                

                    template
                    Mustache template to be used for displaying the combobox dropdown entries.
                    $.fn.TrivialComboBox.icon2LinesTemplate
                

                

                    selectedEntryTemplate
                    Mustacheustache template to be used for displaying the currently selected entry.
                    Same as template
                

                

                    spinnerTemplate
                    Html code for the "loading..." indicator.
                    defaultSpinnerTemplate
                

                

                    noEntriesTemplate
                    Html code for indicating that there is no entry in the dropdown.
                    defaultNoEntriesTemplate
                

                

                    entries
                    The dropdown entries to be displayed initially.
                    []
                

                

                    selectedEntry
                    The initially selected entry.
                    undefined
                

                

                    emptyEntry
                    Fake entry that should be displayed when no entry is selected.
                    {}
                

                

                    queryFunction
                    JavaScript function for retrieving entries depending on the user's input. Takes two parameters:
                        queryString (user input) and
                        resultCallback (should be called with the list retrieved entries).
                    
                    Simple client-side implementation that filters the initially configured array of entries.
                

                

                    autoComplete
                    If true, the component user get auto-completion inside the text field.
                    true
                

                

                    autoCompleteDelay
                    Delay for auto-completion. (This feature may be removed, as there is currently no reason for a delay in auto-completion.)
                    0
                

                

                    showTrigger
                    The trigger is the button-like handle on the side of the combobox. If false: no trigger is shown.
                    true
